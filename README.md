# Hospital Patient Check-In System

A modern 5-step wizard patient check-in system for Sunrise Hospital.

## Setup

```bash
cd hospital-checkin-app
npm install
npm start
```

Open http://localhost:3000

## Features

- 5-step wizard interface
- Patient lookup (returning vs new)
- Auto-fill for returning patients
- Personal information management
- Medical history tracking
- Appointment scheduling
- Insurance verification
- Phone number auto-formatting (xxx-xxx-xxxx)
- Email validation
- Date validation (DOB past, appointments future)
- Progress indicator
- Responsive design
- Elderly-friendly (large fonts, clear buttons)

## Technology

- React 18.2.0
- CSS3
- Form validation
- Mock patient database
